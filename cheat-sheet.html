Basic Bash Commands

Command Description
pwd Print current directory
ls List files in the directory
ls -a List files and directories including hidden files
ls -l Detailed list with permissions, size etc
cd "folder" Change directory to "folder"
cd .. Go up one directory
cd ~ Go to home directory
mkdir "folder" Create a new folder
touch "file" Cerate a new file
rm "file" Delete a file
rm -r "folder" Delete a folder and its contents
mv "source destination" Move or rename files/folders
cp "source destination" Copy files/folders
clear Clear the terminal screen

Basic Version Control (Git) Commands

Command Description
git init Initialize a Git repo
git status Show changes
git add "file" Stage a file
git commit -m "message" Commit stages changes
git push Pull to remote repo
git pull Pull latest changes
